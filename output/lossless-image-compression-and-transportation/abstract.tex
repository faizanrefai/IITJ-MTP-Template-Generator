\begin{flushright}
\huge{\textbf{Abstract}}
\end{flushright}

\vspace{0.5cm}


Many image compression algorithms exist. Some are lossy; others are lossless. Lossless methods allow exact reconstruction of the original image from the compressed data. In this project, we implement and compare several lossless compression techniques. These include classical entropy coding, spatial and transform domain methods, context‑based algorithms, clustering, optimization, and neural network approaches.

We begin with fundamental entropy coding methods. Huffman coding, arithmetic coding, and LZW coding are implemented. These provide a baseline for understanding compression principles. Next, we explore spatial domain methods. Variable block size compression and run‑length encoding (RLE) are used to exploit local redundancy in images. We also implement a lossy‑plus‑residual scheme. In this scheme, a lossy image is transmitted first. The residual difference between the lossy image and the original is then sent. This allows lossless reconstruction.

The JPEG lossless prediction standard is implemented using its seven prediction modes. These modes are based on neighboring pixel values. We then implement two advanced context‑based algorithms: CALIC and LOCO‑I. CALIC uses gradient‑based nonlinear prediction and context modeling of prediction errors. LOCO‑I uses a lower complexity approach based on local gradients. Both achieve good compression.

Transform domain methods are also examined. We implement the S‑Transform, the S+P Transform, and integer wavelet transforms using the lifting scheme. These methods compact energy into fewer coefficients. Clustering‑based compression is implemented for color quantization. Two algorithms are used: K‑Means and Gath‑Geva fuzzy clustering. Gath‑Geva can detect ellipsoidal clusters of varying sizes and densities. Particle swarm optimization (PSO) is then used to find optimal color palettes. Finally, we develop a fuzzy backpropagation neural network (FBPNN). This network combines Gath‑Geva fuzzy memberships with backpropagation learning for adaptive weight updates.

All algorithms are implemented in Python. We use NumPy, Matplotlib, and Pillow. GPU acceleration is provided by JAX and PyTorch on an Apple M4 architecture. Visualizations compare original and compressed images. Compression ratios, MSE, and PSNR are reported. Experimental results show that context‑based methods achieve the best performance. CALIC and LOCO‑I outperform traditional approaches on most test images.
