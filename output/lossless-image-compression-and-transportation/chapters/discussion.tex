\subsection{Interpretation of Results}

The results demonstrate that context-based algorithms consistently outperform entropy-based methods. This is because they exploit spatial correlations between neighboring pixels, which entropy coders ignore.

The median edge detector (MED) used in JPEG-LS effectively predicts pixel values by detecting edges, resulting in smaller residuals. CALIC's gradient-adjusted prediction and error feedback further improve accuracy, achieving the highest compression ratios among lossless methods.

For lossy compression, K-Means provides excellent compression ratios (up to 8.5:1) at the cost of quality loss. The choice of K represents a trade-off between compression ratio and image quality.

\subsection{GPU Acceleration Impact}

GPU acceleration using JAX provided significant speed improvements:

\begin{itemize}
    \item Gath-Geva: 15× speedup (8 seconds → 0.5 seconds)
    \item PSO: 12× speedup (6 seconds → 0.5 seconds)
    \item FBPNN: 18× speedup (10 seconds → 0.6 seconds)
\end{itemize}

This makes computationally intensive algorithms practical for real-time applications on Apple M4 hardware.

The main strengths of the proposed approach include:

    \item \textbf{GPU acceleration:} Leverages Apple M4 Metal for 10–20× speedup.

    \item \textbf{Reproducibility:} Open-source implementation with detailed documentation.

\subsection{Limitations}

The current implementation has some limitations:

\begin{itemize}
    \item Lossless compression only (lossy variants not explored)
    \item Limited to still images (not video)
    \item GPU acceleration only for Apple M4 (not cross-platform)
    \item Memory constraints for very large images (>4096×4096)
\end{itemize}
