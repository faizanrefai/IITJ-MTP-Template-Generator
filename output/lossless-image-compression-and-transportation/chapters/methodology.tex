\subsection{System Architecture}

The proposed system is a modular framework with four layers: data layer, algorithm layer, evaluation layer, and visualization layer. Each algorithm is encapsulated in a separate class with consistent \texttt{compress()} and \texttt{decompress()} methods.

\subsection{Design Approach}

The system follows a modular, object‑oriented design using the Strategy pattern. All algorithms are implemented in Python using NumPy for numerical operations. GPU acceleration is provided via JAX (with Metal backend for Apple M4) and PyTorch MPS.

\subsection{Algorithms Implemented}

The following algorithms were implemented:
\begin{itemize}
    \item Entropy coding: Huffman, Arithmetic, LZW
    \item Spatial methods: RLE, variable block size, lossy+residual
    \item Predictive coding: JPEG lossless (7 predictors)
    \item Context-based: CALIC, LOCO-I
    \item Transform methods: S-Transform, S+P, lifting wavelet
    \item Clustering: K-Means, Gath-Geva (GPU-accelerated)
    \item Optimization: Particle Swarm Optimization (PSO)
    \item Neural network: Fuzzy Backpropagation Neural Network (FBPNN)
\end{itemize}
