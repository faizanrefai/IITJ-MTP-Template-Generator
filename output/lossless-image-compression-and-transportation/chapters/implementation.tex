\subsection{Development Environment}

The entire project was implemented on an Apple M4 MacBook Pro running macOS. The development environment consisted of:

\begin{itemize}
    \item \textbf{Operating System:} macOS Sonoma (14.0+)
    \item \textbf{Programming Language:} Python 3.14
    \item \textbf{IDE:} Visual Studio Code with Python extension
    \item \textbf{Version Control:} Git and GitHub
    \item \textbf{Virtual Environment:} venv for dependency isolation
\end{itemize}

\subsection{Core Libraries Used}

The following open-source libraries were utilized:

\begin{itemize}
    \item \textbf{NumPy (v2.4.2):} For efficient numerical computations and matrix operations
    \item \textbf{Matplotlib (v3.10.0):} For generating all visualizations and plots
    \item \textbf{Pillow (v10.0.0):} For image loading, processing, and manipulation
    \item \textbf{Scikit-learn (v1.5.0):} For K-Means clustering implementation
    \item \textbf{JAX (v0.9.1):} For GPU-accelerated computations on Apple M4
    \item \textbf{PyTorch (v2.5.0):} For Metal Performance Shaders acceleration
    \item \textbf{SciPy (v1.17.1):} For scientific computing and signal processing
\end{itemize}

\subsection{Code Organization}

The implementation follows a modular structure:

\begin{verbatim}
project/
├── algorithms/
│   ├── entropy/          # Huffman, Arithmetic, LZW
│   ├── spatial/          # RLE, Variable Block, Lossy+Residual
│   ├── predictive/       # JPEG Lossless (7 predictors)
│   ├── context/          # CALIC, LOCO-I
│   ├── transform/        # S-Transform, S+P, Lifting Wavelet
│   ├── clustering/       # K-Means, Gath-Geva
│   ├── optimization/     # PSO
│   └── neural/           # FBPNN
├── metrics/              # Compression ratio, MSE, PSNR
├── visualization/        # Plotting functions
└── utils/                # Helper functions
\end{verbatim}

\subsection{GPU Acceleration Implementation}

For computationally intensive algorithms (Gath-Geva, PSO, FBPNN), GPU acceleration was implemented using JAX with Metal backend:

\begin{verbatim}
@jit
def update_centers_jax(X, U, m, c):
    Um = U ** m
    numerator = jnp.dot(Um.T, X)
    denominator = jnp.sum(Um, axis=0).reshape(-1, 1)
    return numerator / (denominator + 1e-10)
\end{verbatim}

This achieved a 10–20× speedup over CPU implementations.
