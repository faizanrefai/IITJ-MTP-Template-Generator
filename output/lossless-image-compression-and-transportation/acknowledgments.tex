\begin{itemize}

    \item \textbf{Matplotlib} for generating comprehensive visualizations and plots
    \item \textbf{Pillow (PIL)} for image processing and manipulation
    \item \textbf{Scikit-learn} for implementing K-Means clustering and comparison
